## Files Included

- `github-stats.html` - Main HTML file with dashboard structure
- `github-stats.css` - Enhanced styles with animations
- `github-stats.js` - JavaScript functionality with animations
- `server.js` - Simple server to serve the files

## How to Run

1. Start the server: `node server.js`
2. Open your browser and navigate to: `http://localhost:8080/`
3. View the animated GitHub stats dashboard
